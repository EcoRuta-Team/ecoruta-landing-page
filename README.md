# EcoRuta

## About the Project
EcoRuta es una solución digital que ayuda a los usuarios a identificar turas saludables con bajos niveles de contaminación.

## Target Segments
- Personas en Lima preocupadas por su salud y el cuidado del medio ambiente
- Deportistas y personas con estilo de vida activo en Lima

## Main Features
- Planeamiento de rutas saludables
- Monitoreo de calidad de aire
- Alertas ambientales
- Comunidad y colaboración
